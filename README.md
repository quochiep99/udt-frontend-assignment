# How to use

## Step 1 :

npm install

## Step 2 :

npm start

## Step 3 :

Open browser and visit http://localhost:3000

# How to run eslint

## To find and list all code style errors

npm run lint

## To auto-correct the possible fixable errors

npm run lint:fix

# How to run test

npm run test
